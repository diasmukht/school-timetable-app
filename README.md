# Timetable Display (local)

Minimal Vite + React app to display rotating class timetables on a monitor.

Getting started (PowerShell):

```powershell
cd "c:\Users\Asus Vivobook\Desktop\sheduleRoza"
npm install
npm run dev
```

Open the URL printed by Vite (usually http://localhost:5173) in the browser. For kiosk/monitor use, open in full screen.

Files of interest:
- `src/components/TimetableDisplay.jsx` — the main component (provided timetable + auto-rotate).
- `index.html`, `src/main.jsx`, `src/index.css` — app entry and styles.
